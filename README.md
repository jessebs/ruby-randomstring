# ruby-randomstring

## Installation
    
To install randomstring, use [rubygems](https://rubygems.org):

    $ gem install randomstring

## Usage

```ruby
require 'randomstring'

Randomstring.generate
# >> "XwPp9xazJ0ku5CZnlmgAx2Dld8SHkAeT"

Randomstring.generate(7)
# >> "xqm5wXX"
```
